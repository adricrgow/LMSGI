Responsive Web Design con Mobile First

El diseño adapta su estructura según el tamaño de pantalla:

  • Móvil (768px): Layout vertical, todo en columna.
  • Tablet (768px - 1023px): Grid de 2 columnas, menú lateral + contenido/aside apilados.
  • Desktop (≥ 1024px): Grid de 3 columnas, todas visibles simultáneamente.

Carga de estilos en cascada

Primero se cargan los estilos base para móvil, luego los específicos para tablet y finalmente los de escritorio.

Estructura de archivos CSS

  • css/movil.css – Estilos base para todos los dispositivos.
  • css/tablet.css – Estilos adicionales para tablets (≥768px).
  • css/desktop.css – Estilos adicionales para escritorio (≥1024px).
